See IDLab in Action

Experience the intuitive interface and powerful features that make ID card creation effortless

Tab 1

Generate IDs

Design and produce printable ID cards from your pupil data and branding assets. Choose photos, output folders, and customize with your logo, coat of arms, motto, and year. Use the preview to confirm layout and colors before exporting as images or PDF.

  • Choose photos/output folders and branding assets
  • Live preview with layout and color confirmation
  • Select categories (grade/form) for targeted export
  • Export as high-quality images or PDF format
Tab 2

Pupil Data Entry

Create, import, edit and maintain the canonical pupil dataset used to populate ID cards. Import Provisional Registers or add pupils manually, then use search, filters and pagination to manage records efficiently with inline editing capabilities.

  • Import Provisional Registers or add pupils manually
  • Search, filter and pagination for easy navigation
  • Inline editing with Add Pupil dialog support
  • Undo/redo functionality for batch edits
Tab 3

ID Photo Tools

Review, match and manage candidate photos so each pupil has the correct image before generating cards. Use the Renaming Wizard to browse gallery thumbnails, assign photos to pupils, and manage matched vs missing photos with export capabilities.

  • Point app to photos folder for automatic detection
  • Renaming Wizard with gallery thumbnail browsing
  • Visual photo-to-pupil assignment with auto-renaming
  • Management view for matched vs missing photo tracking

What is IDLab?

IDLab is a powerful desktop application designed specifically for Zambian Primary and Secondary schools to create professional Candidates ID cards efficiently. Import candidates data, match photos automatically, and generate hundreds of cards with consistent quality.

  • Candidates database with CSV import/export capabilities
  • Smart photo matching by exam numbers
  • Customizable templates with school branding
  • Batch PDF generation for efficient printing
  • Grade and filtering
  • Privacy-focused local data processing
